# Бізнес-контекст skin.one

## Бренди

- **@skin.one.ua** — професійна косметика для обличчя і тіла
- **@skin.one.hair** — професійна косметика для волосся

Усі менеджери працюють з обома брендами.

## Канал і аудиторія

- Instagram Direct (через Sitniks CRM)
- Локація: Черкаси, БЦ Slovianskyi (офлайн-точка + онлайн)
- Аудиторія: жінки 25-55, які інвестують у догляд усвідомлено
- Тип продажу: **експертний** (підбір під тип шкіри/волосся), НЕ транзакційний

## Бренд-імена менеджерів (важливо!)

Усі менеджери підписуються бренд-псевдонімами:
- **"косметолог Анастасія"** — для запитів по обличчю (skin.one.ua)
- **"Катерина"** — для запитів по волоссю (skin.one.hair)

Це навмисний шаблон бренду, НЕ обман. Не вважати помилкою, не штрафувати за тон.

## Робочий час

- 08:00 – 23:00 (Europe/Kiev)
- Клієнтські повідомлення поза цим вікном → менеджер відповість зранку → НЕ вважати ігнором

## Що продається

**Бренди косметики обличчя:** SMART4DERMA, USOLAB, HOLY LAND, CUSKIN, Mediceuticals
**Бренди косметики волосся:** HADAT, RATED GREEN, MAIS, Mediceuticals, CHI

## Що НЕ робимо

- Оптовий продаж
- Постачання косметологам-перепродавцям
- Партнерства з B2B

При таких запитах менеджер коректно відмовляє — це правильна робота, не помилка.

## Товар не в наявності

Якщо товару немає — менеджер пропонує альтернативу. Якщо клієнт відмовляється — менеджер зробив максимум.

## Правила для оцінки Claude

**НЕ позначати `bad`:**
1. Клієнт перестав відповідати після коректного повідомлення менеджера
2. Опт / постачальник / косметолог-перепродавець (відмова без альтернатив — правильно)
3. Товару немає, альтернатива запропонована — клієнт відмовився
4. Конкретний запит на товар X — менеджер відповів ціну без насильного допродажу
5. Швидке привітання + мовчання клієнта

**Ставити `bad` ТІЛЬКИ при явній помилці:**
- Грубість/агресія до зацікавленого клієнта
- Ігнор клієнта у робочий час (08:00-23:00 Київ)
- Дезінформація про продукт
- Менеджер сам відштовхнув готового купити клієнта

**Замовлення оформлене:**
- overall ≥ 4 + експертність ≥ 3 + тон ≥ 5 → `good`
- overall < 3 АБО грубість АБО експертність ≤ 2 → `bad`
- інакше → `neutral`
